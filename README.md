# Modular CI/CD Pipeline
![Build Status](https://github.com/Divyareni/Modular-CI-Pipeline/actions/workflows/pipeline.yml/badge.svg)

This project demonstrates a modular build and automation pipeline using **CMake**, **Python**, and **GitHub Actions**. It mimics real-world hardware/software integration workflows.

## How it Works
1. **CMake**: Handles the C++ compilation and test definitions.
2. **Python**: Orchestrates the build steps and error handling.
3. **GitHub Actions**: Runs the entire pipeline on every push to ensure code quality.
